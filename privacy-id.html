Kebijakan Privasi

Versi 1.0 | Terakhir diperbarui: 22 Desember 2025

Komitmen kami terhadap privasi Anda

Di PillDaddy, kami meyakini informasi kesehatan Anda bersifat pribadi dan harus tetap rahasia. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan aplikasi.

Singkatnya: data sensitif terkait obat tetap berada di perangkat Anda. Anda dapat memilih mengaktifkan sinkronisasi cloud untuk cadangan dan sinkronisasi antar perangkat iOS. Kami menampilkan iklan agar aplikasi tetap gratis.

1. Data yang kami kumpulkan

1.1 Data lokal yang tersimpan di perangkat Anda

Secara default, informasi berikut disimpan di perangkat Anda. Jika Anda mengaktifkan sinkronisasi cloud, data ini akan dienkripsi dan dikirim ke server kami:

  • Nama obat, dosis, dan jadwal
  • Foto atau gambar yang ditambahkan pada obat
  • Catatan atau komentar terkait catatan Anda
  • Riwayat konsumsi obat dan pengingat

1.2 Analitik dan data penggunaan

Kami mengumpulkan analitik anonim untuk meningkatkan aplikasi dan memahami cara penggunaannya:

  • Informasi perangkat (model, versi iOS, versi aplikasi)
  • Statistik penggunaan (fitur yang digunakan, tampilan layar, durasi sesi)
  • Data kinerja (kerusakan, error, waktu muat)
  • Pola interaksi umum (tombol yang diketuk, alur navigasi)
  • Perkiraan lokasi (tingkat negara/wilayah, bukan lokasi presisi)

Data ini dikumpulkan melalui layanan analitik pihak ketiga dan hanya digunakan untuk meningkatkan pengalaman. Tidak termasuk informasi obat atau data medis pribadi Anda.

1.3 Data iklan

Versi gratis PillDaddy didukung oleh iklan. Mitra iklan kami dapat mengumpulkan:

  • Identifikasi perangkat (Advertising ID/IDFA)
  • Alamat IP dan data lokasi umum
  • Interaksi iklan (tayangan, klik)
  • Jenis peramban dan pola penggunaan aplikasi

1.4 Informasi pembelian dalam aplikasi

Jika Anda membeli opsi penghapusan iklan:

  • Pembayaran diproses oleh Apple melalui App Store
  • Kami menerima konfirmasi status pembelian, bukan detail pembayaran Anda
  • Pembelian satu kali dan tidak dapat dikembalikan
  • Penghapusan iklan berlaku pada semua perangkat dengan Apple ID yang sama

2. Di mana data Anda disimpan

2.1 Penyimpanan lokal

Data sensitif terkait obat hanya disimpan di perangkat Anda:

  • Penyimpanan lokal: Informasi obat, jadwal, dan pengingat disimpan secara lokal di iPhone menggunakan mekanisme keamanan iOS.

2.2 Penyimpanan cloud (opsional)

Untuk pengguna yang mengaktifkan sinkronisasi cloud:

  • Sinkronisasi cloud memerlukan login
  • Setelah login, data obat disimpan di server cloud yang aman
  • Sinkronisasi cloud menyediakan cadangan dan sinkronisasi di semua perangkat iOS Anda
  • Sinkronisasi cloud tidak terjadi kecuali pengguna secara eksplisit mengunggah atau mengunduh

3. Cara kami menggunakan informasi Anda

Kami menggunakan data yang dikumpulkan untuk tujuan berikut:

  • Fungsionalitas aplikasi: Menyediakan fitur inti seperti pengingat dan notifikasi
  • Peningkatan: Menganalisis pola penggunaan untuk meningkatkan kinerja dan fitur
  • Pemecahan masalah: Mengidentifikasi dan memperbaiki kerusakan, error, dan masalah teknis
  • Iklan: Menampilkan iklan yang relevan di versi gratis
  • Komunikasi: Mengirim pemberitahuan penting (jika Anda memberikan detail kontak)

4. Layanan pihak ketiga

PillDaddy terintegrasi dengan jenis layanan pihak ketiga berikut:

4.1 Jaringan iklan

Versi gratis menampilkan iklan dari jaringan iklan pihak ketiga, termasuk:

  • Google AdMob atau jaringan serupa
  • Platform mediasi iklan

Jaringan tersebut memiliki kebijakan privasi sendiri terkait pengumpulan dan penggunaan data.

4.2 Layanan cloud (opsional)

Data pengguna yang mengaktifkan sinkronisasi dapat disimpan di server yang berlokasi di Uni Eropa (UE).

5. Berbagi dan pengungkapan data

Kami tidak menjual informasi medis pribadi Anda. Namun, kami dapat membagikan data non-sensitif dalam kasus berikut:

  • Penyedia layanan: Dengan mitra analitik dan iklan seperti dijelaskan di atas
  • Persyaratan hukum: Jika diwajibkan oleh undangâ€‘undang, perintah pengadilan, atau peraturan pemerintah
  • Perubahan perusahaan: Sebagai bagian dari merger, akuisisi, atau penjualan aset
  • Dengan persetujuan Anda: Ketika Anda secara eksplisit mengizinkan berbagi informasi tertentu

Penting: Rincian obat Anda, jadwal, dan data terkait kesehatan TIDAK dibagikan kepada pihak ketiga, kecuali AppWrite (mitra penyimpanan cloud kami saat Anda mengaktifkan sinkronisasi).

6. Integrasi sistem iOS

PillDaddy menggunakan fitur standar iOS agar berfungsi dengan benar:

  • Notifikasi: Kami menggunakan layanan notifikasi iOS untuk mengirim pengingat. Notifikasi ini dibuat secara lokal di perangkat Anda.
  • Cadangan data: Jika cadangan perangkat diaktifkan, data Anda dapat masuk ke cadangan iCloud atau iTunes. Ini dikendalikan oleh pengaturan iOS, bukan oleh PillDaddy.

7. Izin aplikasi

PillDaddy dapat meminta izin iOS berikut:

  • Notifikasi: Diperlukan untuk mengirim pengingat minum obat.
  • Foto/Media (opsional): Akses ke foto atau file di perangkat untuk nada alarm khusus.
  • Pelacakan iklan: Digunakan oleh jaringan iklan untuk menampilkan iklan yang relevan (iklan dapat dinonaktifkan dengan pembelian satu kali).

Anda dapat meninjau dan mengubah izin ini kapan saja di Pengaturan iOS > PillDaddy.

8. Keamanan data

Kami menanggapi keamanan data dengan serius dan menerapkan langkah yang sesuai untuk melindungi informasi Anda:

  • Data lokal: Data yang disimpan di perangkat Anda dilindungi oleh mekanisme penyimpanan aman iOS.
  • Enkripsi: Data yang disinkronkan ke cloud dienkripsi saat transit (TLS/SSL).
  • Kontrol akses: Akses ke data yang tersimpan di server kami dibatasi secara ketat.
  • Praktik terbaik: Kami mengikuti panduan keamanan Apple dan standar industri.

Harap dicatat: Tidak ada metode transmisi melalui internet atau penyimpanan elektronik yang 100% aman. Kami berupaya melindungi informasi Anda, namun tidak dapat menjamin keamanan absolut.

9. Hak dan pilihan Anda

Anda memiliki hak berikut terkait data Anda:

  • Akses: Melihat semua data obat kapan pun dalam aplikasi.
  • Perubahan: Mengedit atau menghapus obat dan pengingat dalam aplikasi.
  • Penghapusan: Menghapus semua data lokal dengan menghapus instalan aplikasi atau mengatur ulang dari Pengaturan.
  • Menonaktifkan iklan: Melakukan pembelian satu kali untuk menghapus iklan, atau batasi pelacakan di Pengaturan iOS.
  • Sinkronisasi cloud: Memerlukan login. Untuk menghapus data cloud, atur ulang data aplikasi kemudian lakukan sinkronisasi â€” data cloud akan ditimpa dengan kumpulan kosong.

Untuk permintaan GDPR/CCPA (akses, penghapusan, portabilitas), hubungi kami menggunakan informasi di bagian akhir kebijakan ini.

10. Retensi data

Kami menyimpan jenis data yang berbeda untuk periode yang berbeda:

  • Data lokal perangkat: Disimpan di perangkat hingga Anda menghapusnya atau menghapus instalan aplikasi.
  • Data sinkronisasi cloud: Disimpan selama akun Anda aktif; dihapus dalam 30 hari setelah penghapusan akun.
  • Data langganan: Disimpan sesuai kebutuhan untuk menyediakan layanan dan memenuhi kewajiban hukum.

11. Privasi anak

PillDaddy tidak ditujukan untuk anak di bawah 13 tahun tanpa pengawasan orang tua:

  • Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak di bawah 13 tahun.
  • Orang tua atau wali dapat menggunakan aplikasi untuk mengelola obat anak mereka.
  • Orang tua harus mengawasi penggunaan aplikasi dan pengaturan perangkat oleh anak.

12. Versi dengan iklan

PillDaddy tersedia gratis dengan iklan:

  • Menampilkan iklan dari jaringan iklan pihak ketiga
  • Mengumpulkan data analitik dan operasional
  • Jaringan iklan dapat menggunakan pengenal iklan Anda untuk iklan yang dipersonalisasi
  • Tersedia sinkronisasi cloud (memerlukan login)
  • Anda dapat melakukan pembelian satu kali untuk menonaktifkan semua iklan

13. Pengguna internasional

PillDaddy tersedia secara global:

  • Penyimpanan data: Data analitik dan cloud dapat disimpan di server di berbagai negara, termasuk AS dan Eropa.
  • Transfer data: Dengan menggunakan aplikasi, Anda menyetujui transfer informasi ke negara dengan undangâ€‘undang perlindungan data yang berbeda.

14. App Store dan pembayaran

Jika Anda membeli opsi penghapusan iklan:

  • Semua pembayaran diproses oleh Apple melalui App Store
  • Kami tidak menerima atau menyimpan informasi pembayaran Anda
  • Kebijakan privasi Apple berlaku untuk pemrosesan pembayaran
  • Kami menerima konfirmasi pembelian untuk menonaktifkan iklan
  • Data transaksi hanya digunakan untuk verifikasi pembelian dan pencegahan penipuan

15. Cookie dan teknologi pelacakan

Meskipun aplikasi tidak menggunakan cookie web tradisional, kami menggunakan teknologi serupa:

  • Pengenal seluler: Pengenal perangkat dan ID iklan untuk iklan
  • Penyimpanan lokal: Untuk menyimpan preferensi dan data obat di perangkat Anda

Anda dapat mengatur ulang pengenal iklan di pengaturan iOS untuk membatasi pelacakan.

16. Perubahan kebijakan privasi ini

Kami dapat memperbarui kebijakan privasi ini secara berkala. Saat itu terjadi:

  • Versi yang diperbarui akan dipublikasikan di aplikasi dan situs web kami.
  • Nomor versi dan tanggal "Terakhir diperbarui" akan diperbarui.
  • Melanjutkan penggunaan aplikasi setelah perubahan berarti Anda menerima kebijakan yang diperbarui.
  • Versi terbaru selalu tersedia di pengaturan aplikasi atau situs kami.

17. Hubungi kami

Jika Anda memiliki pertanyaan atau masukan tentang kebijakan privasi ini atau cara PillDaddy menangani data Anda, hubungi kami:

  • Situs: saltukalakus.com
  • Pengembang: Saltuk Alakus

Untuk permintaan GDPR/CCPA (akses, penghapusan, portabilitas), sebutkan "Privacy Request" di subjek email.

Ringkasan: Data sensitif terkait obat tetap berada di perangkat Anda. Kami menampilkan iklan untuk mendukung pengembangan. Anda dapat menghapus iklan dengan pembelian satu kali.
